程式本體在Main資料夾,編譯指令如下

編譯

javac --module-path 你的sdk的lib位置 --add-modules javafx.controls,javafx.media -encoding UTF8 Main.java

執行

java --module-path 你的sdk的lib位置 --add-modules javafx.controls,javafx.media Main
